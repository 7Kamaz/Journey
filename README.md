CTF‑THM‑MrRobot

Este repositório contém soluções e anotações do CTF “Mr Robot” da plataforma TryHackMe — um desafio prático de captura de bandeiras (Capture The Flag, CTF) focado em técnicas de pentest e segurança ofensiva. 

📌 Objetivo

O objetivo deste CTF é hackear a máquina Mr Robot, encontrar as três flags escondidas e documentar o processo de exploração, desde enumeração até exploração e escalonamento de privilégios. 

🧠 Sobre o desafio

O Mr Robot CTF é um desafio de segurança nível iniciante a intermediário na plataforma TryHackMe. A máquina simula um ambiente real vulnerável para que você pratique técnicas comuns de segurança ofensiva. 

Ele inclui etapas como:

Conexão à VPN e deploy da máquina

Enumeração de portas e serviços

Identificação de vulnerabilidades em serviços web

Exploração com bruteforce e obtenção de shells

Escalonamento de privilégios


🛠 Tecnologias e Ferramentas utilizadas

Durante a resolução deste CTF foram usadas (entre outras):

Kali Linux / Parrot / VM de análise

nmap (enumeração de portas)

Gobuster / dirb / nikto (enumeração de diretórios)

Hydra / Burp Suite (ataques de força bruta e análise de requisições)

Exploração de aplicações web

Reverse shell / pós‑exploração

🚀 Como usar

1. Clone este repositório:

git clone https://github.com/7Kamaz/CTF-THM-MrRobot.git
cd CTF-THM-MrRobot


2. Abra o arquivo MR. Robot.md para ver o passo a passo detalhado da resolução.


3. Veja a pasta report/ para conferir as flags encontradas e explicações.



🧩 Flags

As três flags encontradas durante o CTF estão documentadas no relatório (report/) com contexto técnico e o que foi necessário para chegar nelas.
