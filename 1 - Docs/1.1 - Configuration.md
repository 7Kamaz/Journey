# Configuration.VMs
Aqui eu vou explicar um pouco sobre a configuração feitas das VM's e os requisítos mínimos para este projeto
------------------------------------------------------------------
* Detalhamento dos Requisitos do sistema:
Componenente          | RAM   | Disco| Observação
Kali Linux (VM)       | 2GB   | 20GB | (Menos que 2GB deixa a interface gráfica lenta. 20GB é o mínimo para instalar e atualizar ferramentas básicas.)
Metasploitable 2 (VM) | 512MB | 8GB  | (*Essa VM não tem interface gráfica e é muito leve. *O disco virtual ocupa pouco espaço real (aprox. 1-2GB), mas reserva cerca de 8GB.)
Windows (Host)        | 4GB   | N/A  | (O Windows precisa de cerca de 4GB livres para ele mesmo rodar sem travar enquanto gerencia as VMs.TOTAL MÍNIMO~6.5 GB~30 GB Livres.)

------------------------------------------------------------------

* Downloads Necessários:

VirtualBox: https://www.virtualbox.org/wiki/Downloads

Kali Linux: https://www.kali.org/get-kali/#kali-virtual-machines (Escolha a versão VirtualBox)

Metasploitable 2: https://sourceforge.net/projects/metasploitable/files/Metasploitable2/

-----------------------------------------------------------------------------------------------

**Passo 1: Configurar a Rede no VirtualBox
    Antes de criar as VMs, vamos preparar a rede:

    Abra o VirtualBox

    Vá em Arquivo → Preferências → Rede

    Clique na aba Rede Interna

    Clique no ícone + (criar nova rede)

    Nomeie como "lab-interno" e clique em OK

---

** Passo 2: Instalar e Configurar o Kali Linux
    Instalação:

    Clique em Nova no VirtualBox

    Nome: Kali

    Tipo: Linux

    Versão: Debian (64-bit)

    Memória: 4096 MB (4GB)

    Disco rígido: Criar novo disco → VDI → Dinamicamente alocado → 25 GB

    Configuração de Rede:

    Clique na VM do Kali → Configurações

    Vá em Rede

    Aba Adaptador 1:

    Conectado a: Rede Interna

    Nome: lab-interno

    Clique em OK

    Primeira Inicialização:

    Inicie a VM do Kali

    Login: kali / Senha: kali

    Abra o terminal e digite: ´´´sudo apt update && sudo apt upgrade -y´´´

---

**Passo 3: Configurar o Metasploitable 2
    Importação:

    No VirtualBox, clique em Arquivo → Importar Appliance

    Clique no ícone de pasta e selecione o arquivo .ova do Metasploitable 2

    Clique em Next e depois Import

    Configuração de Rede:

    Clique na VM do Metasploitable → Configurações

    Vá em Rede

    Aba Adaptador 1:

    Conectado a: Rede Interna

    Nome: lab-interno

    Clique em OK

    Inicialização:

    Inicie a VM do Metasploitable

    Login: msfadmin / Senha: msfadmin

---

**Passo 4: Verificar a Conexão
    No Kali Linux, abra o terminal e digite:

    ´´´
    # Ver seu IP
        ip addr show

    # Ver IP do Metasploitable (substitua 192.168.56.1 por sua rede)
        nmap -sn 192.168.56.1/24

    # Testar conectividade
        ping 192.168.56.101
    ´´´

    IPs Típicos:
    Kali Linux: 192.168.56.102
    Metasploitable: 192.168.56.101

---

**Passo 5: Configurações Adicionais Úteis
    No Kali Linux:

    ´´´
    # Instalar ferramentas adicionais
        sudo apt install medusa nmap enum4linux

    # Criar pasta do projeto
        mkdir meulaboratorio
        cd meulaboratorio
    ´´´

    No Metasploitable:

    ´´´
    # Ver serviços ativos
        netstat -tulpn

    # Verificar se DVWA está rodando
        curl http://localhost/dvwa
    ´´´

------------------------------------------------------------------

✅ Verificação Final
Se tudo deu certo, você deve conseguir:

 Fazer ping do Kali para o Metasploitable

 Acessar o DVWA no navegador do Kali: http://192.168.56.101/dvwa

 Conectar via SSH: ssh msfadmin@192.168.56.101
